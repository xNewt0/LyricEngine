# 🎵 LyricEngine

**LyricEngine** is a lightweight and fast lyric retrieval engine designed for developers.  
It provides an easy way to search, fetch, and process song lyrics programmatically.

Built for personal usage and music-related tools.

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/xNewt0/LyricEngine.git
cd LyricEngine
npm run dev
